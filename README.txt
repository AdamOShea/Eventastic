A full-stack cross-platform mobile app that aims to aggregate social
and cultural event listings and suitable flights/accommodation
together in one user-friendly interface.

Planned Features include:
• Event Aggregation: Collects and displays social and cultural event
  listings spanning a range of categories, including music, arts,
  sports, and festivals

• Travel and Accommodation Integration: Aggregates suitable
  flights and accommodation options for listed events. Simplifies
  process of planning trips for events by presenting travel options
  and estimated prices alongside event information

• In-App Event hosting and Ticketing: Provides a platform for users to
  host events with built-in QR code-based ticketing options

• And More: Other planned features if time allows include user
  friend’s lists, real-time chat, event forums, event notifications and
  more
